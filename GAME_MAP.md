# WORLD MAKER — Game Map
# This file is a cheat sheet so the AI always knows where things are.
# When you want to change something, use these exact words!

## THINGS YOU CAN CHANGE WITH THE AI:

| What you say       | What it changes                            | Code Label          |
|--------------------|--------------------------------------------|---------------------|
| "ground color"     | The color of the floor/ground              | [GROUND_COLOR]      |
| "sky color"        | The background sky color                   | [SKY_COLOR]         |
| "player speed"     | How fast the player walks/runs             | [PLAYER_SPEED]      |
| "jump height"      | How high the player jumps                  | [JUMP_HEIGHT]       |
| "other players"    | What color other players look like         | [OTHER_PLAYER_COLOR]|

## TIPS:
- Colors use hex codes. Common ones:
  - Dark Red:   0x8B0000
  - Dark Green: 0x003300
  - Dark Blue:  0x00008B
  - Black:      0x000000
  - White:      0xFFFFFF
  - Purple:     0x4B0082
  - Orange:     0xFF4500

- Speed & Jump: higher number = faster/higher

## HOW TO USE:
Just type naturally in the launcher chat! Examples:
  ✅ "make the ground dark blue"
  ✅ "make the player speed 800"
  ✅ "make the jump height 600"
  ✅ "change the sky to black"
